# Herramienta Transfer

La herramienta **Transfer** es un script diseñado para copiar información, parámetros o elementos entre modelos de Revit de forma rápida y segura, manteniendo el estándar de HMV.

---

## 🎯 ¿Para qué sirve?

Esta función es ideal cuando necesitas:
* Migrar configuraciones de un proyecto antiguo a uno nuevo.
* Copiar parámetros compartidos de forma masiva.
* Estandarizar la información sin hacerlo manualmente.

---

## 🚀 Modo de Uso

Sigue estos pasos para ejecutar el comando correctamente:

1. Abre el modelo de Revit destino (donde quieres que llegue la información).
2. Ve a la cinta superior y selecciona la pestaña **HMV Tools**.
3. En el panel de utilidades, haz clic en el botón **Transfer**.
4. En la ventana emergente, selecciona el modelo de origen y marca las casillas de lo que deseas transferir.
5. Haz clic en **Aceptar**.

## ⚠️ Consideraciones Importantes

!!! warning "Precaución antes de usar"
    Es muy recomendable sincronizar o guardar tu modelo local antes de hacer transferencias masivas, por si necesitas deshacer los cambios.

!!! tip "Consejo de productividad"
    Si seleccionas elementos específicos antes de hacer clic en el botón, la herramienta solo transferirá información a esos elementos.
